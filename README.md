# TT00BT65-3001-capstone-project
Simple string manipulation program written in c and using curses for tui.

## Features
The program allows the user to perform the following operations on a string:
```
A)  Count the number of vowels in the string
B)  Count the number of consonants in the string
C)  Convert the string to uppercase
D)  Convert the string to lowercase
E)  Display the current string
F)  Enter another string
G) Read string from file
H) Write string to file

M) Display this menu 
X) Exit the program
```
See the [project specification](docs/project-specification.md) for more details.
