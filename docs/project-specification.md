# Specifications for the Capstone Project
This document outlines the needed features for the capstone project.

## Project Overview
The project should be cut up into multiple functions that each perform a specific task.
Large overview of needed functions:
* Function to count vowels in a string
* Function to count consonants in a string
* Function to convert string to uppercase
* Function to convert string to lowercase
* Function to display the current string
* Function to enter a new string
* Function to read string from file
* Function to write string to file
* Function to display the menu
